# pretty-git-diff

## Why：

  - Windows 上的 `vimdiff`（a.k.a. `vim -d file1 file2`）有点 bug
  - [Computed Diff - Diff Checker](https://www.diffchecker.com/diff) 很好用，但是命令行版本要注册
  - 我需要这个东西，实在是太需要了！

## Usage：

  - `./pgd.sh file1 file2`
  - 就这么简单（可以先 `make` 但不必须）
